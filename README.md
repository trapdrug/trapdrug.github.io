## TrapDrug Portal
# trapdrug.github.io

Это мой первый сайт-переходник (портфолио), который служит центральной точкой для связи и демонстрации моих проектов.

## О проекте

**TrapDrug Portal** — это минималистичный веб-сайт, созданный для того, чтобы:
*   Представлять меня как разработчика (TrapDrug).
*   Служить удобным переходником (`переходником`) на мои другие ресурсы.
*   Демонстрировать мои первые шаги в веб-разработке.

Это мое портфолио, где собраны все важные ссылки и контакты.

## Содержание сайта

На данный момент на сайте представлены следующие элементы:

*   **Заголовок:** `portal / trap_drug` — идентификация проекта и автора.
*   **Основной контент:**
    *   Текст, приветствующий посетителей.
    *   Ссылка на профиль в мессенджере (`Message/DM @enkipo`).
    *   Ссылка на GitHub (`моё портфолио`), где можно увидеть мои проекты.
*   **Дополнительно:**
    *   Ссылка для поддержки (`Пожертвование на развитие`).
